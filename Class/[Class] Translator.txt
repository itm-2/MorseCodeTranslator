(인터페이스 실현까지 모두 설명함)

translator는 주어진 문자열을 다른 문자열로 변역합니다.

TranslationToAlnum
모스부호를 숫자 또는 알파벳으로 번역합니다.

모스부호에는 Q부호도 포함되며
모스부호는 SPACE를 기준으로 분류합니다.

다음 허프만 코드에 따라 진행합니다.
0: dit
10: dah
11: SPACE

그러면, translator는 다음을 시도합니다.

1. 11만 존재한다 -> SPACE 문자(띄어쓰기)로 번역
2. dit, dah 신호를 해석하여 대응되는 알파벳으로 번역
3. 입력되는 string size가 10 이상인 경우 '0' 또는 Q부호로 번역 시도
4. 1 - 3까지 해당 사항 없음 -> UI에 warnInvalid 요청(0을 반환)

참고로, 모든 문자 끝에는 구분자의 의미로 SPACE가 포함됩니다.

e.g.)
01011 -> 'A'
0001011 -> 'V'
11 -> ' '
10100101001011 -> "QTA"

TranslationToMorse
알파벳 또는 숫자를 모스부호로 번역합니다.

알파벳과 숫자는 ASCII 형태로 주어집니다.
이때, 해당 ASCII 코드에 대응되는 모스부호를
이전의 허프만 코드 형식으로 번역합니다.

'A' -> 01011
'V' -> 0001011
